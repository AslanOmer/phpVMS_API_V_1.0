# phpVMS_API_V_1.0

This module is to publish certain phpVMS site data into a JSON file for third party use.

Install:

1. Download the repository.
2. Unzip the repository.
3. Upload the files in folder's order.
4. Access it by "adminurl('/api')"
5. Enjoy!
